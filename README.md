# React with webpack development environment

Reactとstylusをwebpackで動かす開発環境をとりあえず作ったので置いておく...:ant::ant::ant:

ブログ:http://www.twopipe.com/front-end/2014/12/22/react-webpack-env.html

# Usage

webpackをインストール
```
$ npm i webpack -g
```

通常のcompile

```
$ webpack
```
or

ローカルサーバをたてる

```
$ npm run start
```
